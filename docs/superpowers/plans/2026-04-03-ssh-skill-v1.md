# SSH Skill V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个基于 Go 和 OpenSSH 的 Codex SSH skill，支持跳板链、审计、隧道、动态代理和长任务托管。

**Architecture:** 代码分为 CLI 入口、配置与主机清单、SSH 参数生成、执行器、审计、运行时状态和长任务管理七个模块。底层全部复用系统 `ssh`，本地只负责参数拼装、状态管理与审计记录，避免自实现 SSH 协议。

**Tech Stack:** Go 1.22、OpenSSH、TOML、JSONL

```go
func Resolve(inv model.Inventory, cfg model.Config, alias string) (model.ResolvedHost, error) {
    host, ok := inv.Hosts[alias]
    if !ok {
        return model.ResolvedHost{}, fmt.Errorf("host not found: %s", alias)
    }
    return model.ResolvedHost{
        Alias: alias,
        Host: host.Host,
        User: firstNonEmpty(host.User, cfg.DefaultUser),
        Port: firstNonZero(host.Port, cfg.DefaultPort),
    }, nil
}
```
